Static site + CSV CMS

Edit the CSV. Upload the photos. Refresh the page.

This page reads all of its content from content.csv. Each row becomes one story card and one map pin. To change the site, edit the CSV fields or drop photos into the matching folder inside images/.

1. Add a row Create or edit a row in content.csv with the title, copy, date, and coordinates you want to show.
2. Upload photos Put images in images/your-folder/ and name them 1.jpg, 2.jpg, 3.jpg, or list exact paths in the CSV.
3. Refresh The HTML file fetches the CSV directly, so your next refresh shows the new content layout.
